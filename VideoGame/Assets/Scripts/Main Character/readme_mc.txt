what each script is doing: 

Basic/arrow.cs - basic behavior for the arrow
Basic/controls.cs - player movement (run/jump/dash)
Basic/healPotion.cs - heal item behavior
Basic/ladderMechanic.cs - ladder behavior
Basic/player_health.cs - player health behavior
Basic/shoot.cs - shoot arrows behavior

Inventory/itemInventory.cs - checking which item inventory is full and has what
Inventory/pickUp.cs - picking up the item and placing the icon in the inventory for use
Inventory/Slot.cs - dropping the item from the inventory
Inventory/Spawn.cs - spawning the item in the world after dropping it from the inventory
